# LC236 二叉树的最近公共祖先

## 思路
- 后序递归：在左右子树中查找目标节点，命中数量决定当前节点是否为最近公共祖先。
- 递归出口：遇到 `None` 返回 `None`；遇到 `p` 或 `q` 直接返回当前节点，向上传递命中信息。
- 合并规则：左右都命中则当前节点为 LCA；仅一侧命中则把那侧继续上交；两侧皆空返回 `None`。
- 使用 `Rc::ptr_eq` 判等，避免值相同但节点不同的错误；树节点用 `Rc<RefCell<_>>` 方便共享与修改子指针。

## 图示
示例一（返回 3）：
```
        3  ← 最近公共祖先
       / \
      5   1
     / \ / \
    6  2 0  8
      / \
     7   4
p = 5, q = 1
```

示例二（返回 5）：
```
        3
       / \
      5  ...
     / \
    6   2
       / \
      7   4  ← 与 5 的最近公共祖先是 5 自身
p = 5, q = 4
```

## 代码要点
- `lowest_common_ancestor` 递归合并返回值，实现后序判定。
- `build_tree_with_map` 将层序数组转换为二叉树，并返回值到节点的映射，便于快速取出 `p`、`q`。
- 示例 `main` 复现题目给出的三个样例，输出对应的 LCA。

## 复杂度
- 时间复杂度 O(n)，每个节点仅被访问一次。
- 空间复杂度 O(h)，递归栈深度与树高度一致。
